# Responsive Travel Landing Page

## Task 1 - SkillCraft Technology Internship

This project is a fully responsive travel landing page developed using HTML, CSS, and JavaScript.

## Features

- Responsive Navigation Bar
- Fixed Navigation Menu
- Navigation Color Change on Scroll
- Hover Effects
- Smooth Scrolling
- Hero Section
- Popular Destinations
- Tour Packages
- Testimonials
- Booking Form
- Contact Section
- Responsive Design
- Scroll Animations
- Counter Animation

## Technologies Used

- CSS3
- JavaScript
